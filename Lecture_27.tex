\intertext{$c_n$ is obtained from an odd periodic extension of the initial condition}  
c_n &= \frac{2}{2} \left[\int_0^1 0.1 x \sin\left( \frac{n \pi}{2} x  \right) dx  + \int_1^2  0.1 (2-x) \sin\left( \frac{n \pi}{2} x  \right) dx \right] \\
&=0.1\left[  \frac{4 \sin\left( \frac{n\pi}{2} \right) -2n \pi \cos\left( \frac{n\pi}{2} \right)    }{\pi^2 n^2} + \frac{4 \sin\left( \frac{n\pi}{2} \right) +2n \pi \cos\left( \frac{n\pi}{2} \right)    }{\pi^2 n^2}  \right]\\
&= \frac{0.8}{n^2 \pi^2} \sin\left( \frac{n\pi}{2} \right) }
